# chat_palette
webhookを用いてチャットへ投稿するためのパレットを提供するシンプルなGUIツール。

## 使い方
### 準備
「環境設定」で、名前、webhookを設定する。
### パレットの追加
テキストを入力し、追加ボタンを押す。
### テキスト投稿
パレットボタンを押すと、そのまま投稿できる。
